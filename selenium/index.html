Selenium Overview


  • Automation tool for web based application built originally by Thoughtworks.
  • The core engine of Selenium is developed in JavaScript.
  • Programming Language support: Java, C#, Perl, PHP, Python, Ruby
  • Cross Browser Support: IE, Firefox, Opera, Safari and Google Chrome.
  • Cross Platform support: Windows, Linux , Macintosh
  • Cannot automate desktop based applications

Selenium IDE


  • Integrated development environment for Selenium tests
  • Enables you to record a browser session.
  • Implemented as Mozilla Firefox add-on.
  • Allows you to record, edit and debug tests.
  • Auto detection of web elements using id, name, xpath.
  • Ability to export the IDE code into Java, C#, Perl, PHP, Python, Ruby

Selenium Webdriver


  • Webdriver is a API that interacts with the browser directly.
  • Selenium webdriver makes direct calls to the browser using each browser’s native support for automation.
  • Can extend the API to add custom functionality.

Object locating strategies


  • By.id(“id of Object”)
  • By.linkText(“Text Used In Link”)
  • By.partialLinkText(“part of the link”)
  • By.tagName(“HTMLNodeType”)
  • By.className(“cssClassOnTheElement”)
  • By.cssSelector(“cssSelectorToTheElement”)
  • By.xpath(“Xpath of the eleemnt”)
  • By.name(“nameOfElement”)
